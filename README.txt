Red Hawk has been developed in an environment with the following features:

Kali Linux (2021.2)
Django (2.2.22)
Python (3.9.2)

Red Hawk is an application with the aim of providing the community with a tool to automate pentesting tasks in web applications, providing the user with academic 
knowledge or web vulnerability analysis results in a business environment.

You are free to modify, at any time, the source code of Red Hawk.

Use Red Hawk only against web applications that you have permission to pentest or that you own. The developer is in no way responsible for any misuse for which the tool is used.

To use Red Hawk:
python3 manage.py runserver
and then, web GUI will be available in http://localhost:8000

Enjoy.
